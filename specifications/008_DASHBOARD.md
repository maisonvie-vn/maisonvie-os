# 🏛️ MVOS Dashboard Specifications (008)

## 1. Visual Aesthetics
The user interface implements the French Neoclassical dark-mode styling system.

### Color Tokens
- **Background (Nền)**: `#102B2A` (deep forest green)
- **Panels/Sidebar (Phụ)**: `#042726` (dark forest green)
- **Primary Text**: `#ffffff` (white text)
- **Accents (Accent Brass)**: `#A8884E` (brass gold tone)
- **Secondary / Hover Gold**: `#C59D5B`

### Typography
- **Headers**: `Cormorant Garamond` (editorial serif)
- **Body & Controls**: `Inter` (sans-serif)

## 2. Interactive Modules
- **Navbar**: Sticky navigation menu containing links to Dashboard, Documents, SOP, and Admin pages. Includes a live connection status indicator.
- **Seating Layout Chart**: Lists tonight's seating queue, VIP tables, party sizes, and arrival statuses.
- **Agent Feed**: Live feed displaying autonomous execution logs.
