# 🌐 MVOS Language Policy

This document establishes the language standards for all codebase, user interface, database schema, and documentation layers within the Maison Vie Operating System (MVOS).

---

## 📋 Standards Table

| Layer | Language Standard | Target Audience | Example / Scope |
| :--- | :--- | :--- | :--- |
| **Frontend UI** | **Vietnamese** | Staff & Operators | Labels, buttons, headings, statuses |
| **Internal SOP** | **Vietnamese** | Kitchen & FOH Staff | Standard Operating Procedures manuals |
| **Codebase** | **English** | AI & Human Developers | Types, variables, functions, comments |
| **Database** | **English** | Database Admins | Table names, keys, column attributes |
| **API Routes** | **English** | Integration Engines | `/api/v1/reservations`, `/api/auth` |
| **Technical Docs** | **English** | Systems Architects | Architecture diagrams, rule definitions |
| **Operational Docs**| **Vietnamese** | FOH Operations Lead | Admin configurations, staff guidelines |
